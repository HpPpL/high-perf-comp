# Task 2: Анализ масштабируемости

## Описание

Исследование масштабируемости параллельной реализации решения уравнения теплопроводности при различных конфигурациях.

## Параметры исследования

- **Количество процессов**: 1, 2, 4, 8, 16, 24
- **Количество точек (N)**: 10000, 25000, 50000
- **Конечное время**: T = 10⁻⁴ (уменьшено для ускорения)

## Компиляция

```bash
make
```

## Запуск

```bash
# Локально с указанием количества процессов и размера сетки
mpirun -np 4 ./heat_equation 10000

# На кластере (запускает все тесты автоматически)
sbatch run_task2.sbatch
```

## Результаты

Результаты сохраняются в файл `scalability_results.csv` в формате:
```
num_procs,N,min_time,max_time,avg_time
```

Для анализа результатов используйте скрипт `analyze_results.py` из корневой папки hw3.

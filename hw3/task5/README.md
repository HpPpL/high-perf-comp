# Task 5: Гибридная задача (MPI + GPU)

## Описание

Реализация решения уравнения теплопроводности с использованием гибридного подхода: 2 MPI процесса, каждый из которых выполняет вычисления на своем GPU.

## Особенности

- Использование CUDA для вычислений на GPU
- Каждый MPI процесс работает со своим GPU
- Обмен граничными значениями между процессами через MPI
- Использование pinned memory для оптимизации передачи данных CPU ↔ GPU

## Требования

- CUDA Toolkit
- MPI с поддержкой CUDA
- 2 GPU (или 2 процесса на одном GPU с разными device ID)

## Компиляция

```bash
make
```

## Запуск

```bash
# Локально (требуется ровно 2 процесса)
mpirun -np 2 ./heat_equation 10000

# На кластере
sbatch run_task5.sbatch
```

## Примечание

Цель этой задачи - не ускорение программы, а изучение компиляции и написания гибридных программ (MPI + CUDA).

# Task 3: Асинхронная передача данных

## Описание

Реализация решения уравнения теплопроводности с использованием асинхронных операций MPI (`MPI_Isend`/`MPI_Irecv`) для перекрытия вычислений и передачи данных.

## Особенности

- Использование `MPI_Isend` и `MPI_Irecv` для асинхронной передачи граничных значений
- Перекрытие вычислений внутренних точек с передачей данных
- Использование `MPI_Wait` для синхронизации перед вычислением граничных точек

## Компиляция

```bash
make
```

## Запуск

```bash
# Локально
mpirun -np 4 ./heat_equation 10000

# На кластере
sbatch run_task3.sbatch
```

## Результаты

Результаты сохраняются в файл `scalability_results_async.csv` для сравнения с синхронной версией из Task 2.

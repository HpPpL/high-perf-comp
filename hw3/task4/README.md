# Task 4: Коллективные операции

## Описание

Реализация решения уравнения теплопроводности с использованием коллективных операций MPI для распределения начальных данных и сбора результатов.

## Используемые коллективные операции

- `MPI_Bcast` - распространение информации о распределении данных между процессами
- `MPI_Scatterv` - распределение начального распределения температуры
- `MPI_Gatherv` - сбор финальных результатов на root процессе
- `MPI_Reduce` - сбор статистики производительности

## Компиляция

```bash
make
```

## Запуск

```bash
# Локально
mpirun -np 4 ./heat_equation 10000

# На кластере
sbatch run_task4.sbatch
```

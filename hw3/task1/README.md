# Task 1: Проверка корректности алгоритма

## Описание

Реализация параллельной версии алгоритма решения одномерного уравнения теплопроводности с проверкой корректности путем сравнения с точным решением.

## Параметры задачи

- `k = 1.0` - коэффициент температуропроводности
- `h = 0.02` - шаг по пространству
- `τ = 0.0002` - шаг по времени
- `T = 0.1` - конечное время
- `u₀ = 1.0` - начальная температура

## Точное решение

Для проверки корректности используется точное решение:

```
u(x,t) = (4u₀/π) * Σ[m=0 to ∞] (1/(2m+1)) * exp(-kπ²(2m+1)²t/l²) * sin(π(2m+1)x/l)
```

## Компиляция

```bash
make
```

## Запуск

```bash
# Локально
mpirun -np 1 ./heat_equation

sbatch run_task1.sbatch
```

# Task 6: 2D Теплопроводность с MKL

## Описание

Реализация решения двумерного уравнения теплопроводности с использованием Intel Math Kernel Library (MKL) для эффективных операций с матрицами.

## Уравнение

Двумерное уравнение теплопроводности:
```
∂u/∂t = k * (∂²u/∂x² + ∂²u/∂y²)
```

Конечно-разностная схема:
```
u_(i,j)^(n+1) = u_(i,j)^n + (kτ/h²) * (u_(i+1,j)^n + u_(i-1,j)^n + u_(i,j+1)^n + u_(i,j-1)^n - 4*u_(i,j)^n)
```

## Граничные условия

Дирихле: u = 0 на всех границах области [0,1]×[0,1]

## Требования

- Intel MKL библиотека
- Компилятор с поддержкой C++17

## Компиляция

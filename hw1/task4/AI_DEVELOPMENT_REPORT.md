## Сгенерированный результат

ИИ создал полноценную реализацию задачи, включающую:

### 1. Структура проекта
- `task4.h` - заголовочный файл с объявлениями функций и структур данных
- `task4.hpp` - реализация алгоритмов умножения матриц и функций профилирования
- `task4.cpp` - основная программа с тестированием и бенчмарками
- `Makefile` - система сборки с поддержкой различных конфигураций
- `run_task4.sbatch` - SLURM скрипт для запуска на кластере
- `README.md` - документация проекта

### 2. Реализованные алгоритмы
- **Sequential** - последовательная реализация (базовая линия)
- **OpenMP Basic** - базовая параллельная реализация с `#pragma omp parallel for`
- **OpenMP Collapse** - оптимизированная версия с `collapse(2)` и `simd`
- **OpenMP Schedule** - версия с динамическим планированием `schedule(dynamic, 16)`

### 3. Система профилирования
- Структуры `ProfileResult` и `DetailedProfile` для хранения результатов
- Функция `profile_function()` для измерения времени выполнения
- Функция `detailed_profile_omp_multiply()` для детального анализа
- Сохранение результатов в CSV формате
- Расчет GFLOPS и анализ масштабируемости

### 4. Особенности реализации
- Кроссплатформенность (macOS/Linux)
- Автоматическое определение доступности OpenMP
- Поддержка произвольного количества потоков
- Оптимизация с флагами компилятора `-O3`
- Использование современных C++11 возможностей

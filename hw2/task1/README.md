# Task 1: CUDA Matrix Multiplication with Global Memory

## Описание

Реализация умножения матриц на GPU с использованием глобальной памяти CUDA. Матрицы передаются через pinned memory для оптимизации передачи данных между CPU и GPU.

## Особенности

- Использование глобальной памяти GPU
- Pinned memory для передачи данных CPU ↔ GPU
- Двумерная организация блоков и потоков (16x16)
- Измерение времени с помощью `cudaEventElapsedTime`
- Верификация результатов сравнением с CPU версией

## Компиляция

### На кластере:
```bash
module load nvidia_sdk/nvhpc/23.5
make
```

### Локально (если CUDA установлена):
```bash
make
```

## Запуск

### Локально:
```bash
./gemm [N]
```
где N - размер матрицы (по умолчанию 1024)

### На кластере через SLURM:
```bash
sbatch run_task1.sbatch
```

### Интерактивно на кластере:
```bash
srun -n 1 --gpus=1 -A proj_1447 ./gemm 1024
```

## Структура кода

- `gemm.cu` - основной файл с CUDA kernel и host кодом
- Kernel `matrixMultiplyGlobal` - выполняет умножение матриц на GPU
- Используется column-major порядок хранения матриц

## Результаты

Программа выводит:
- Время выполнения (мс)
- Производительность (GFLOPS)
- Результаты верификации

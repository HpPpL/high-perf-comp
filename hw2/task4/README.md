# Task 4: cuBLAS Matrix Multiplication

## Описание

Использование библиотеки cuBLAS для умножения матриц. cuBLAS - это высокооптимизированная библиотека NVIDIA для линейной алгебры.

## Особенности

- Использование `cublasDgemm` для умножения матриц
- Автоматическая оптимизация на уровне библиотеки
- Поддержка различных операций (transpose, etc.)
- Максимальная производительность

## Функция cuBLAS

```c
cublasDgemm(handle, CUBLAS_OP_N, CUBLAS_OP_N,
            N, N, N,
            &alpha,
            d_A, N,
            d_B, N,
            &beta,
            d_C, N);
```

Где:
- `C = alpha * A * B + beta * C`
- `CUBLAS_OP_N` - без транспонирования
- `alpha = 1.0`, `beta = 0.0` для нашего случая

## Компиляция и запуск

```bash
module load nvidia_sdk/nvhpc/23.5
make
./gemm [N]
```

## Преимущества cuBLAS

- **Максимальная производительность**: использует все оптимизации GPU
- **Простота использования**: одна функция вместо написания kernel
- **Надежность**: протестированная и оптимизированная библиотека
- **Поддержка различных операций**: transpose, hermitian, etc.

## Ожидаемые результаты

cuBLAS обычно показывает лучшую производительность по сравнению с самописными реализациями благодаря:
- Оптимизации на уровне ассемблера
- Использованию tensor cores (на поддерживаемых GPU)
- Оптимальному использованию памяти

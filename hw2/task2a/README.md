# Task 2a: Unified Memory

## Описание

Реализация умножения матриц с использованием Unified Memory (Unified Memory). Unified Memory позволяет использовать единое адресное пространство для CPU и GPU без явного копирования данных.

## Особенности

- Использование `cudaMallocManaged` для выделения unified memory
- Нет явного переноса данных между CPU и GPU
- Автоматическое управление памятью CUDA runtime
- Доступ к данным как с CPU, так и с GPU

## Компиляция и запуск

Аналогично Task 1:
```bash
module load nvidia_sdk/nvhpc/23.5
make
./gemm [N]
```

## Преимущества Unified Memory

- Упрощение кода (нет явных cudaMemcpy)
- Автоматическая миграция данных
- Единое адресное пространство

## Ограничения

- Требует compute capability 6.0+ для оптимальной работы
- Может быть медленнее явного управления памятью для больших матриц
